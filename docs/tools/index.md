# Security Tools

Various tools are available to help secure your Web3 applications.

## Categories

- Static Analysis Tools
- Dynamic Analysis Tools
- Testing Frameworks
- Monitoring Tools
- Development Tools

## Popular Tools

- Slither: Static analysis for Solidity
- Mythril: Security analysis framework
- Foundry: Testing framework
- Hardhat: Development environment
- Tenderly: Transaction simulation and monitoring
