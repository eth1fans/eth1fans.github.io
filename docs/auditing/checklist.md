# 安全审计清单

在准备或进行安全审计时使用此清单。

## 访问控制

- [ ] 实现了适当的基于角色的访问控制
- [ ] 所有者函数受到保护
- [ ] 没有未授权的访问路径
- [ ] 关键操作使用多重签名

## 输入验证

- [ ] 所有输入都经过验证
- [ ] 地址零检查
- [ ] 整数溢出/下溢保护

## 重入保护

- [ ] 遵循检查-效果-交互模式
- [ ] 在需要的地方实现重入保护
- [ ] 在状态更新后进行外部调用

## 错误处理

- [ ] 适当的错误消息
- [ ] 失败的交易正确回滚
- [ ] 没有静默失败
- [ ] 为重要操作发出事件
